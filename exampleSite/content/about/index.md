## What Is This?

This is the demo site for **hugo-cyberdelic**, a Hugo theme that brings cyberdelic visual design to
static websites. The theme combines the warmth of 1960s psychedelic concert posters with the cold
precision of cyberpunk futurism.

## Features

- **Dark-first design** — Violet-black backgrounds with neon accent colors
- **Fluid typography** — Responsive type scale using `clamp()`
- **Design tokens** — CSS custom properties for consistent styling
- **Glitch effects** — CSS-only glitch text animations
- **Accessibility** — Skip links, ARIA labels, keyboard navigation, focus indicators
- **Responsive** — Looks great on any device

## Usage

To use this theme in your own Hugo site:

```bash
git init
git submodule add https://github.com/magnus919/hugo-cyberdelic.git themes/hugo-cyberdelic
echo 'theme = "hugo-cyberdelic"' >> hugo.toml
hugo server -D
```
